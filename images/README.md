This folder holds all your portfolio images.

HOW TO USE:
-----------

1. Save your artwork images here
2. Recommended format: JPG or PNG
3. Recommended size: 1200x1200px or larger
4. Compress images before adding (use tinypng.com)
5. Name files: portfolio-01.jpg, portfolio-02.jpg, etc.

EXAMPLE:
--------

After adding images, your portfolio page will reference them like:
  <img src="images/your-artwork.jpg" alt="Description">

Update the src="" paths in portfolio.html to match your image filenames.
